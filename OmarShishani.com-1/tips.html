How to write an annoy-o-matic loop

You may think that the annoy-o-matic can be accomplished with an 'if' statement, but if you use only the 'if' statement, the annoy-o-matic will run only once, and then the code will exit. To properly run the annoy-o-matic, you need to have a while loop, so that the code will keep running until it receives the answers 'yeah' or 'yes'.
